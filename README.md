# mDowód — Klon demonstracyjny (1:1)

Projekt demonstracyjny React + Vite odwzorowujący ekran "mDowód" z możliwością wgrania zdjęcia.
Dołączone zdjęcie zostało skopiowane z przesłanego pliku (neutralne).

## Uruchomienie

1. `npm install`
2. `npm run dev`
3. Otwórz http://localhost:5173

Plik ZIP przygotowany automatycznie.
